# Makes K come first
grade_order = ['K', '1', '2', '3', '4', '5']
def set_grade_order(df, grade_col='Grade'):
    df[grade_col] = pd.Categorical(df[grade_col], categories=grade_order, ordered=True)
    return df.sort_values(grade_col)

Reading Log Progression Trends (Weeks 1 → 4)¶

Objective: Track student and class performance across the four reading logs (RL1 RL4) to identify growth patterns, stagnation, or areas needing attention.

Key Takeaways:

  • Classes showing steady upward trends indicate consistent improvement in student reading performance.
  • Classes with flat or declining trends may require targeted instructional support.
  • Comparing classes within a grade allows benchmarking and sharing of best practices among teachers and administrators.
  • Patterns across grades can reveal systemic trends in early reading development.

SQL Template¶

In [130]:
query_template = """
SELECT
    Teacher AS Class,
    SUM(RL1) AS RL1,
    SUM(RL2) AS RL2,
    SUM(RL3) AS RL3,
    SUM(RL4) AS RL4
FROM MM_DATA
WHERE Grade = '{}'
GROUP BY Teacher
ORDER BY Teacher
"""

Generating & Plotting Per-Grade Trends¶

  • For each grade, total points for each reading log (RL1RL4) are aggregated per class.
  • Data is reshaped from wide to long format to facilitate comparative plotting across classes.
  • Line charts are generated per grade to visualize how each class progresses over the four reading logs.
In [131]:
%%capture
grades = MMRC_df['Grade'].unique()
grade_names = {'K':'K','1':'1st','2':'2nd','3':'3rd','4':'4th','5':'5th'}
trends_dfs = {}

for g in grades:
    df = %sql {query_template.format(g)}
    df = df.DataFrame()

for g, df in trends_dfs.items():
    df_long = df.melt(

    n_classes = df['Class'].nunique()

    plt.grid(True)
    plt.legend(bbox_to_anchor=(1.05,1), loc="upper left")

Class Performance Distribution & Outliers

Objective: Examine how student performance varies within and across classes, and identify outliers to highlight exceptional or struggling students.

Key Takeaways:

  • Provides a clear picture of class and teacher-level performance variability.
  • Helps administrators and teachers identify students needing support and recognize top performers.
  • Combines average points and outlier detection to support data-driven decisions for instruction and intervention.

Average Points Per Student Per Class¶

  • Computes the mean total points per student for each class.
  • Provides an overview of typical student performance and helps compare classes at a glance.
  • Insight: Classes with higher averages indicate stronger overall performance, while lower averages may signal areas needing instructional support.
  • Visual: Boxplots showing the distribution of average TotalPoints per Student across Classes.

Identify Outliers Using IQR¶

  • Detects outliers using the Interquartile Range (IQR) method.
In [134]:
Q1 = MMRC_df['TotalPoints'].quantile(0.25)
Q3 = MMRC_df['TotalPoints'].quantile(0.75)
IQR = Q3 - Q1
outliers = MMRC_df[(MMRC_df['TotalPoints'] < Q1 - 1.5*IQR) | (MMRC_df['TotalPoints'] > Q3 + 1.5*IQR)]
